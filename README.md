# АИС Интернет-провайдера (Курсовой проект)

Это веб-приложение для управления абонентами интернет-провайдера с системой аутентификации и авторизации на основе ролей.

## Технологический стек

- **Бэкенд:** FastAPI (Python)
- **База данных:** PostgreSQL (без ORM)
- **Безопасность:** JWT-токены, passlib (bcrypt), http-only cookie
- **Фронтенд:** HTML, Bootstrap 5, HTMX

## Пошаговая инструкция по запуску

### Шаг 1: Подготовка окружения

1.  Установите **Python 3.8+** и **PostgreSQL**.
2.  Создайте и активируйте виртуальное окружение.

### Шаг 2: Установка зависимостей

1.  Создайте файл `requirements.txt` и скопируйте в него содержимое из раздела ниже.
2.  Установите все библиотеки:
    ```bash
    pip install -r requirements.txt
    ```

### Шаг 3: Настройка базы данных

1.  Создайте базу данных `provider_db` в PostgreSQL.
2.  Выполните скрипт `database/schema.sql` для создания таблиц.
3.  Выполните скрипт `database/seed.sql` для добавления тестовых пользователей и данных.

### Шаг 4: Конфигурация приложения

1.  В корне проекта создайте файл `.env`.
2.  Скопируйте в него содержимое из раздела ниже.
3.  **ОБЯЗАТЕЛЬНО:** Сгенерируйте свой собственный секретный ключ командой `openssl rand -hex 32` в терминале и вставьте его в `JWT_SECRET_KEY`.
4.  Укажите ваши данные для подключения к PostgreSQL.

### Шаг 5: Запуск приложения

1.  Выполните команду: `uvicorn src.main:app --reload`
2.  Откройте в браузере **http://127.0.0.1:8000**. Вас должно перенаправить на страницу входа.
3.  Используйте данные из `seed.sql` для входа: `admin` / `admin_pass`, `manager` / `manager_pass`, `tech` / `tech_pass`.```
