Требования к формату файла

Файл должен быть в кодировке UTF-8 и содержать JSON-массив (список) объектов, где каждый объект представляет одного абонента. Пример:

[
    {
        "full_name": "Иванов Иван Иванович",
        "address": "г. Минск, ул. Центральная, д. 1, кв. 5",
        "phone_number": "+375291112233",
        "balance": 15.50
    },
    {
        "full_name": "Петрова Мария Сергеевна",
        "address": "г. Минск, пр. Независимости, д. 10",
        "phone_number": "+375334445566",
        "balance": -5.20
    }
]

Обязательным является только поле full_name. Остальные поля (address, phone_number, balance) опциональны.
